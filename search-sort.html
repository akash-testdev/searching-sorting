Searching & Sorting

This page contains questions, answers and programs for searching and sorting.

Logic - Each element of the input array is compared with the element to be searched. If an array element matches the element to be searched, the position of the element is returned, else the search continues till the last element of the array. If the element to be searched is not present in the array -1 is returned. 

public int linearSearch(int a[], int ele)
{
    int n = a.length;
    for(int i=0; i<n; i++)
    {
        if(a[i] == ele)
            return i+1;
    }
    return -1;
}
