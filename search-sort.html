Searching & Sorting

This page contains questions, answers and programs for searching and sorting.

Logic - Each element of the input array is compared with the element to be searched. If an array element matches the element to be searched, the position of the element is returned, else the search continues till the last element of the array. If the element to be searched is not present in the array -1 is returned.

Initially, we take 2 pointers, left and right. The left pointer points to the first element and the right pointer points to the last element of the array. The right pointer remains fixed at the last element of the array. Each array element pointed by the left pointer is compared with the element to the searched,

- If the match happens, the index of the left pointer is returned.
- Else the left pointer is incremented to the next array element.

This process gets repeated till the left pointer is <= right pointer.

public int linearSearch(int a[], int ele)
{
  int rightPointer = a.length-1;
  for(int leftpointer=0; leftpointer<=rightPointer; leftpointer++)
  {
    if(a[leftpointer] == ele)
      return leftpointer+1;
  }
  return -1;
}

Note: Since we are moving into the concept of thinking everything in terms of pointers, notice that rightPointer is not kept equal to the length of the array, it is made to point to the last array element by keeping it equal to the length of the array minus 1. In the loop, we have taken care, that the left pointer goes all the way till it is equal to the right pointer and not ends one element before the right pointer.

Logic - The first thing to remember is Binary search works only when the array is sorted. Initially we take 2 pointers, left and right pointing to the first and last element of the input array. Till left <= right, we determine the middle element between the left and right pointers and compare it with the element to be searched.

- In case the search element is equal to the middle element, we return the index of middle element.
- In case the search element is greater than the middle element, we focus on the right part of the array by incrementing the left pointer to mid+1.
- In case the search element is lesser than the middle element, we focus on the left part of the array by decrementing the right pointer to mid-1.

This process gets repeated till left <= right. 

public int binarySearch(int a[], int ele)
{
  //applicable only on sorted arrays
  int leftPointer = 0;
  int rightPointer = a.length-1;
  int midPointer;
  while(leftPointer<=rightPointer)
  {
    midPointer = (leftPointer + rightPointer)/2;
    if(ele > a[midPointer])
      leftPointer = midPointer + 1;
    else if(ele < a[midPointer])
      rightPointer = midPointer - 1;
    else
      return midPointer+1;
  }
  return -1;
}

Logic - In bubble sort, every element of the array is compared to the next element. If the next element is smaller then the elements are swapped. In one complete iteration, the largest element in this way bubbles to the last position in the array. This process is repeated again starting from the first element of the array till the second last element. We do nnot consider the last element since we already know that it is the largest array element and has reached its desired position in the array.

From the coding point of view, we maintain two pointers, left pointer and the right pointer initially pointing to the first and last array elements. With each iteration, the element at the left pointer is compared with the element next to it and in case the next element is samller, the elements are swapped and the left pointer is incremented. The left pointer continues to move towards the right till it reaches the right pointer. This is when the largest array element has reached the last index inn the array. The right pointer is now moved one position towards the left, the left poinnter is reset back to the first element of the array and the process is repeated till the left pointer again reaches the right pointer. This entire process is repeated N times, where N is the number of elements in the input array. 

public int[] bubbleSort(int a[])
{
  int rightPointer = a.length-1;
  int n = a.length;
  for(int i=0; i<n; i++)
  {
    int leftPointer=0;
    while(leftPointer<rightPointer)
    {
      if(a[leftPointer] > a[leftPointer+1])
      {
        int temp = a[leftPointer];
        a[leftPointer] = a[leftPointer+1];
        a[leftPointer+1] = temp;
      }
      leftPointer++;
    }
    rightPointer--;
  }
  return a;
}

Logic - In insertion sort, we start from the second array element and try to place it at its correct position in the subarray on the left so that the left array always remains sorted. This process is repeated from the second element till the last element of the array. This is called insertion sort because the element picked up is inserted at is correct position in the left subarray.

From the coding point of view, we maintain two pointers, left pointer and the right pointer pointing to the first and last array elements. These left and right pointers remain static throughtout execution. We also maintain two more pointers, current pointer initially pointing to the second array element and previous pointer pointing to the position just before the position pointed by current pointer. The current pointer points to the element that has to be placed at its correct position in the left subarray (from 0 till currentPointer-1). The previous pointer compares the element at its index with the element at current pointer and if the element at current pointer is smaller than the element at previous pointer, we copy the element at previous pointer to the poisition next to the previous pointer and move the previous pointer one index left. This process is repeated till either the previous pointer reaches the first index of the array or till we encounter an element in the previous pointer whose value is smaller than the element at current pointer. In both these cases, we place the element pointed by current pointer to the position next to the position pointed by the previous pointer. This entire process is repeated till the current pointer reaches the last index of the array. 

public int[] insertionSort(int a[])
{
  int leftPointer = 0;
  int rightPointer = a.length-1;
  int currentPointer = 1;
  while(currentPointer<=rightPointer)
  {
    int elementToBePlacedAtCorrectPosition = a[currentPointer];
    int previousPointer = currentPointer-1;
    while(previousPointer>=leftPointer && elementToBePlacedAtCorrectPosition<a[previousPointer])
    {
      a[previousPointer+1] = a[previousPointer];
      previousPointer--;
    }
    a[previousPointer+1] = elementToBePlacedAtCorrectPosition;
    currentPointer++;
  }
  return a;
}
